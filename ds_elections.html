🔄 Import Votes from Email Sync

Each vote from any device is emailed as a sync packet. Paste the email body (or multiple emails) below and click Import — the system will extract and merge all votes automatically.

How to collect votes from other devices:

  1. Open your Gmail at demograhyelections2026@gmail.com
  2. Search for emails with subject starting DSE-SYNC|
  3. Open each email, copy the full body, paste it above
  4. You can paste many emails together — duplicates are ignored
  5. Click Import & Merge

📋 Setup Instructions (one-time, 5 min)

  1. Go to sheets.google.com → create a new spreadsheet
  2. Click Extensions → Apps Script
  3. Delete all default code, paste the DSE_AppsScript.gs file contents
  4. Click Deploy → New Deployment → Web App
  5. Set Execute as: Me & Who has access: Anyone
  6. Click Deploy → copy the Web App URL
  7. Paste it in the field above and click Save & Test
